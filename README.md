# 实验室易耗品管理系统
## 系统需求概述

* 系统有两类用户：普通用户和管理员
* 管理员负责易耗品的入库、普通用户的管理，申请单的审核；易耗品的类别管理。
* 普通用户在线填写领用申请单，申请单通过审核后不能删除。
* 易耗品数据、申请单的导出
* 统计用户领用易耗品情况
## 项目环境
jdk 1.8+， mysql 5.6+ Tomcat 8.5
## 技术选择
* java+SpringMVC+Mybatis+jQuery
## 项目用途
* 可以作为SpringMVC框架学习
## 项目部署
* 导入到ide ，给他服务器环境就可以了，扩展什么的都上传的， 不需要额外导入lib，数据库文件在./sql。
